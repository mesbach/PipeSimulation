# 🌊 Water Flow Simulation v1
🧠 Water Flow Simulation v1 is a comprehensive software project designed to simulate and analyze water flow in various hydraulic systems. The project aims to provide a user-friendly and interactive environment for engineers, researchers, and students to model, simulate, and visualize water flow in different scenarios. The simulation is built using a range of technologies, including .NET Framework, WPF, and custom libraries.

## 🚀 Features
- **Simulation Environment**: A interactive and customizable simulation environment to model and analyze water flow in various hydraulic systems.
- **Pipe and Valve Modeling**: Support for modeling different types of pipes and valves, including straight pipes, bend pipes, sudden expansion and contraction pipes, and valves.
- **Energy Loss Calculation**: Calculation of energy loss in pipes and valves based on various factors, such as flow velocity, pipe diameter, and valve type.
- **Visualization**: Real-time visualization of the simulation, including pipe and valve geometry, flow velocity, and energy loss.
- **User Input and Interaction**: Support for user input and interaction, including setting simulation parameters, inserting nodes and edges, and adjusting pipe and valve properties.

## 🛠️ Tech Stack
* **Frontend**: WPF (Windows Presentation Foundation)
* **Backend**: .NET Framework
* **Database**: None (in-memory data storage)
* **AI Tools**: None
* **Build Tools**: Visual Studio
* **Libraries and Frameworks**: System, System.Collections.Generic, System.Linq, System.Text, System.Windows, System.Windows.Controls, System.Windows.Media, GDI.constanta

## 💻 Usage
1. **Launch the Application**: Launch the Water Flow Simulation v1 application
2. **Create a New Simulation**: Create a new simulation by selecting the simulation type and setting the simulation parameters
3. **Insert Nodes and Edges**: Insert nodes and edges to create the hydraulic system model
4. **Adjust Pipe and Valve Properties**: Adjust pipe and valve properties, such as diameter, length, and type
5. **Run the Simulation**: Run the simulation and visualize the results
